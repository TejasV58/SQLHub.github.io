DATA MANIPULATION LANGUAGE

The SQL commands that deals with the manipulation of data present in the database belong to DML or Data Manipulation Language and this includes most of the SQL statements.Data Manipulation Language (DML) statements are used for managing data in database. DML commands are not auto-committed. It means changes made by DML command are not permanent to database, it can be rolled back.

Here are some commands that come under DML:
  • INSERT
  • DELETE
  • UPDATE

DML Commands

1.INSERT

INSERT command is used to insert data in the table .

Syntax

INSERT INTO table_name VALUES(data1, data2, ...);

Example

Consider a table student with the following fields.

St_id Name age
INSERT INTO student VALUES(101, 'Raghav', 15);

The above command will insert a new record into student table.

St_id Name age
101 Raghav 15

Insert value into only specific columns

We can use the INSERT command to insert values for only some specific columns of a row. We can specify the column names along with the values to be inserted

Syntax

INSERT INTO table_name(column1, column2, ...) VALUES(value1, value2, ...);

table_name: name of the table.
column1: name of first column, second column …
value1, value2, value3 : value of first column, second column,… for the new record

Example

Consider the following table Student,

St_id Name age DOB
1 Raghav 17 16-jan-2002
2 Suresh 16 23-nov-2001
INSERT INTO Student (St_id, Name, DOB) VALUES ('5' , 'PRATIK' , '12-jan-2000');

After executing the following command it will insert a record in the Student table in the specified column.Here in this example the data was enetered into the 'St-id', 'Name' and 'DOB' columns and the remaining column was empty.

St_id Name age DOB
1 Raghav 17 16-jan-2002
2 Suresh 16 23-nov-2001
5 PRATIK NULL 12-jan-2000

Insert NULL value to a column

NULL value can be inserted into a table in two ways.They are:

Example

INSERT INTO Student(St_id, name) VALUES(10, 'Atharva');

The above command will insert only two column values and the other column is set to null.

OR,

INSERT INTO Student VALUES(10,'Atharva', null, null);

Output:

St_id Name age DOB
1 Raghav 17 16-jan-2002
5 PRATIK NULL 12-jan-2000
10 Atharva NULL NULL

The NULL value are inserted in the places where the value is set as NULL or in the other method where the specific column is not specified that column is set as NULL.


Insert Default value to a column

Default value can be inserted into a table as shown,

Suppose the column age in our tabel has a default value of 20.

INSERT INTO Student VALUES(103,'Nisha', default, '13-feb-2000')

Also, if you run the below query, it will insert default value into the age column, whatever the default value may be.

INSERT INTO Student(St_id, Name, DOB) VALUES(103 ,'Nisha', '13-feb-2000' )

Output:

St_id Name age DOB
101 Raghav 17 16-jan-2002
102 PRATIK NULL 12-jan-2000
103 Nisha 20 13-feb-2000

2.DELETE

DELETE command is used to delete data from a table.

Syntax

DELETE FROM table_name;

Let's take a sample table student:

St_id Name age DOB
101 Rajiv 18 16-jan-2002
102 Akash 20 12-jan-2000
103 Nitin 20 23-dec-2000

Delete all Records from a Table

DELETE FROM Student;

The above command will delete all the records from the table student.

St_id Name age DOB
 

Delete a particular Record from a Table

In our student table if we want to delete a single record, we can use the WHERE clause to provide a condition in our DELETE statement.

DELETE FROM student WHERE St_id=103;

The above command will delete the record where St_id is 103 from the table student

St_id Name age DOB
101 Rajiv 18 16-jan-2002
102 Akash 20 12-jan-2000

Delete multiple Records from a Table

In our student table if we want to delete a multiple record, we can use the WHERE clause to provide a condition in our DELETE statement.

DELETE FROM student WHERE age=20;

The above command will delete the records where age are 20 from the table student

St_id Name age DOB
101 Rajiv 18 16-jan-2002


3.UPDATE

UPDATE command is used to update any record of data in a table. We can update single columns as well as multiple columns using UPDATE statement as per our requirement.

General Syntax

UPDATE table_name SET column_name = new_value WHERE some_condition;

table_name : name of the table where you want to make changes.
column_name : name of the column where you want modify.
new_value : new value of that you want to insert or change.
some_condition : any condition based on which you want to make change.
WHERE is used to add a condition to any SQL query.

Consider the table Customer,

Cust_id Name age Registration_date
1001 Sanjay 30 16-aug-1990
1002 Nilesh 23 19-june-1995
1003 Namrata 28 23-dec-1992

Update a particular Record from a Table

UPDATE Customer SET age=25 WHERE Cust_id=1002;
Cust_id Name age Registration_date
1001 Sanjay 30 16-aug-1990
1002 Nilesh 25 19-june-1995
1003 Namrata 28 23-dec-1992

In the above statement, if we do not use the WHERE clause, then our update query will update age for all the columns of the table to 25.


Updating Multiple Columns

We can also update values of multiple columns using a single UPDATE statement.

UPDATE Customer SET Name='Abhi', age=23 where Cust_id=1001;
Cust_id Name age Registration_date
1001 Abhi 23 16-aug-1990
1002 Nilesh 25 19-june-1995
1003 Namrata 28 23-dec-1992

The above command will update two columns of the record which has Cust_id 1001.
